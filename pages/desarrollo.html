Desarrollo de este sitio web

El presente sitio web fue realizado para mostrar lo aprendido en el curso de Desarrollo Web de Coderhouse. Como tal, se encuentra en construcción y permanente actualización en base a los contenidos nuevos adquiridos. Para el diseño y desarrollo web existen diferentes lenguajes, que nos permiten llevar el diseño en papel (sketch) a una estructura que pueda interpretar un computador. En este sitio se emplearon dos lenguajes bases: HTML y CSS.

HTML

Es un "lenguaje" de marcado de etiquetas. Se comenzó con el uso de html para crear los documentos para web, es decir darle la estructura al sitio.

CSS

Luego se utilizó CSS, el cual es un lenguaje de diseño gráfico para brindarle estilo al sitio, es decir permite controlar la apariencia de una página web.

Bootstrap

Es un framework que permite crear interfaces web con CSS y JavaScript. Permite lograr sitios responsive de forma sencilla. En este sitio web se usó Bootstrap para maquetar el sitio en version desktop y mobile -además de media queries- y para realizar el navbar del header.

GIT y GitHub

En este sitio se utilizó el controlador de versiones GIT para crear un repositorio local que posteriormente se subió a GitHub para crear un repositorio remoto y así implementar la corrección por parte de los tutores de Coderhouse.

Javascript

Se implementará en un futuro cercano Javascript, el cual es un lenguaje de programación que permite añadir características interactivas al sitio web, y de esta manera por ej se podrá enviar el formulario de contacto del sitio.
